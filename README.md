# 医护职业转型测试系统

## 项目介绍

这是一个为医护人员设计的职业转型测试系统，基于医疗行业大数据和职业发展模型，帮助医护人员评估自身优势、识别转型方向，并提供个性化的职业发展建议。

## 项目特色

1. **专业评估**：基于医疗行业数据分析
2. **智能建议**：个性化转型路径推荐
3. **发展路径**：清晰的职业发展路线图
4. **女性互助**：公益自发的女性自主项目，所有开发人员均为女性

## 使用说明

1. 打开 `index.html` 开始测试
2. 填写个人信息（当前职位、工作年限、学历等）
3. 完成10个能力评估问题
4. 查看个性化的转型建议和行动计划

## 测试流程

1. **个人信息**：填写基本信息
2. **能力评估**：回答10个专业问题
3. **结果分析**：获得转型匹配度和推荐方向
4. **行动计划**：获取具体的发展建议

## 技术特点

- 纯前端实现，无需后端服务器
- 响应式设计，支持移动设备
- 本地存储，保护用户隐私
- 简洁美观的UI设计
