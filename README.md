# 医护职业转型智能测试系统

## 项目概述

这是一个基于Web的医护职业转型智能测试系统，帮助医生、护士和医技人员评估个人技能和兴趣，匹配最适合的职业转型方向。

## 系统特点

### 🎯 专业数据支持
- 基于真实的医疗行业转型数据
- 涵盖医生、护士、医技三大群体共19类专业
- 8个转型方向的详细属性分析

### 🧠 智能算法评估
- 采用条件筛选+评分模型的三步算法
- 动态权重调整机制
- 多维度综合评估

### 📊 个性化报告
- 详细的转型方向推荐
- 个性化的行动计划
- 风险分析与应对策略

### 🔒 隐私保护
- 所有数据在浏览器本地处理
- 不上传服务器，保护用户隐私
- 可本地保存和下载报告

## 算法原理

### 三步评估流程
1. **条件筛选**: 根据经济压力、风险承受、学习意愿、地理位置等条件排除不合适的方向
2. **偏好加权**: 根据个人技能兴趣和转型偏好调整方向权重
3. **综合评分**: 在剩余方向中计算综合匹配度，生成个性化推荐

### 评分维度
- **基础匹配度**: 基于专业数据的初始匹配
- **技能匹配度**: 个人技能与方向需求的匹配
- **兴趣匹配度**: 个人兴趣与工作内容的匹配
- **风险适配度**: 个人风险承受与方向风险的匹配

## 使用说明

### 1. 本地运行
```bash
# 克隆或下载项目
cd medical-career-test

# 使用Python启动本地服务器
python3 -m http.server 8000

# 或直接打开HTML文件
open index.html
```

### 2. 测试流程
1. **基本信息收集** (5题): 职业类型、科室、工作年限等
2. **技能评估** (7题): 技能水平、兴趣偏好、转型期望等
3. **转型准备评估** (6题): 经济压力、风险承受、家庭支持等
4. **生成报告**: 个性化推荐、行动计划、风险分析

### 3. 功能特性
- **进度保存**: 浏览器本地保存测试进度
- **动态问题**: 根据职业类型动态显示科室选项
- **响应式设计**: 支持手机、平板、电脑访问
- **报告下载**: 可下载完整的文本报告
- **结果分享**: 支持分享测试结果

## 数据来源

### 专业数据
- **医生群体**: 8类专科医生的转型方向数据
- **护士群体**: 6类护理岗位的转型方向数据
- **医技群体**: 5类医技人员的转型方向数据

### 转型方向
1. **医疗科技**: 医疗设备、软件、AI等科技公司
2. **医药企业**: 制药、医疗器械、医药流通等企业
3. **健康管理**: 健康管理公司、体检中心、康养机构
4. **教育培训**: 医学院校、培训机构、在线教育
5. **互联网医疗**: 互联网医疗平台、数字健康公司
6. **政府/事业单位**: 卫健委、疾控中心、医保局等
7. **深造研究**: 继续深造、科研机构、高校
8. **自由职业**: 自由执业、咨询顾问、个人工作室

## 开发说明

### 数据扩展
如需添加新的职业类型或转型方向，请修改：
- `js/questions.js`: 问题数据和方向数据
- `js/algorithm.js`: 算法逻辑和匹配规则

### 样式定制
- 修改 `css/style.css` 调整界面样式
- 使用CSS变量统一颜色主题
- 响应式断点: 768px, 1024px

### 算法优化
- 可在 `CareerTransitionAlgorithm` 类中调整权重参数
- 可扩展新的筛选条件和评分维度
- 可集成机器学习模型提升预测准确性

## 隐私与安全

### 数据保护
- 所有测试数据仅在用户浏览器中处理
- 不上传服务器，不收集个人身份信息
- 报告可本地保存，不会泄露给第三方

### 使用限制
- 测试结果仅供参考，不构成职业建议
- 最终决策请结合个人实际情况
- 系统不保证100%的准确性
