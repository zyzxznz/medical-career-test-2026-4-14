# 医护职业转型测试系统 - 项目总结

## 📋 项目概述

一个完整的Web应用，帮助医护人员评估自身优势、识别转型方向，并提供个性化的职业发展建议。系统基于医疗行业大数据和职业发展模型，为医护人员提供科学的转型指导。

## 📊 评估算法

### 评分系统
1. **问题权重**：不同问题对不同转型方向有不同的权重
2. **用户意向**：用户选择的转型意向获得额外加权
3. **归一化处理**：将原始分数转换为0-100%的匹配度

### 推荐算法
1. **多维评分**：计算用户在6个方向的综合得分
2. **排序筛选**：按得分排序，取前3名作为推荐
3. **个性化调整**：根据用户背景微调推荐结果
