# 部署指南

## 🚀 快速部署

### 方法一：直接使用（最简单）
1. 复制整个 `medical-career-test` 文件夹
2. 在任何地方打开 `index-fixed-fixed.html` 文件

### 方法二：本地服务器
```bash
# 进入项目目录
cd /path/to/medical-career-test

# 运行启动脚本
./start.sh

# 或手动启动
python3 -m http.server 8000
```

### 方法三：在线部署
1. 上传到 GitHub，启用 GitHub Pages
2. 部署到 Netlify、Vercel 等静态网站托管服务
3. 部署到自己的 Web 服务器

## 🌐 网络部署

### GitHub Pages
1. 创建 GitHub 仓库
2. 上传所有文件
3. 设置 Settings → Pages → Source → main branch
4. 访问 `https://[username].github.io/[repository]`

### Netlify
1. 拖拽文件夹到 Netlify 网站
2. 或连接 GitHub 仓库
3. 自动部署完成

### 自有服务器
1. 上传文件到服务器 `public_html` 或 `www` 目录
2. 确保服务器支持静态文件服务
3. 配置域名指向
